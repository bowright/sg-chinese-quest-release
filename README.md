# SG Chinese Quest Releases

This repository is the public distribution channel for Android release builds of SG Chinese Quest.

The application source code is maintained in a separate private repository. This public repository is intentionally kept minimal and only contains:
- release APK files
- checksum files
- release notes
- public-facing documentation

## Downloads

Download the latest APK from the [Releases](../../releases) page.

Current published file name:
- `sg-chinese-quest.apk`

## Verify the APK

Each release includes a `SHA256SUMS.txt` file. You can verify the downloaded APK before installing it.

Example on Linux or macOS:

```bash
sha256sum sg-chinese-quest.apk
```
