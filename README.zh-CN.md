# SG Chinese Quest 发布仓库

这个仓库用于公开发布 SG Chinese Quest 的 Android 安装包。

应用源码保存在另一个 private 仓库中。这个 public 仓库会尽量保持精简，只放：
- APK 安装包
- 校验文件
- 发布说明
- 面向下载者的公开说明文档

## 下载

请到 [Releases](../../releases) 页面下载最新版 APK。

当前发布文件名：
- `sg-chinese-quest.apk`

## 校验 APK

每个 release 都会附带 `SHA256SUMS.txt`。下载安装包后，建议先核对 SHA256 再安装。

Linux / macOS 示例：

```bash
sha256sum sg-chinese-quest.apk
```
